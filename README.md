# Fake News Detection Web Application

## Overview
This project implements a machine learning-based system to classify news articles as fake or real using Natural Language Processing (NLP).

## Technologies Used
- Python
- scikit-learn
- TF-IDF Vectorization
- Logistic Regression

## How It Works
1. Dataset is loaded from an Excel file.
2. Text data is preprocessed and cleaned.
3. TF-IDF is used to convert text into numerical features.
4. A Logistic Regression model is trained.
5. The model predicts whether a news article is fake or real.

## Results
The model achieved high accuracy during testing. The results are influenced by the structured nature of the dataset.
